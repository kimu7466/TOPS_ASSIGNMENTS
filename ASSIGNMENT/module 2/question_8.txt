Q. 8) In how many ways can a CSS be integrated as a web page? 

Ways CSS Can Be Integrated in a Web Page:
External CSS: Create a separate .css file and link it in the HTML file using <link> tag.
Internal CSS: Include CSS within the <style> tag inside the <head> section of HTML.
Inline CSS: Apply CSS directly to an HTML element using the style attribute.
